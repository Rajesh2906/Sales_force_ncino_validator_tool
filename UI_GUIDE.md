### Responsive Breakpoints

```css
/* Desktop (> 1024px) */
- Side-by-side layout
- 50% width each panel
- Full height split

/* Tablet (вүӨ 1024px) */
- Stacked vertical layout
- Full width, 50% height each
- SyntaxValidator on top
- Preview on bottom

/* Mobile (вүӨ 768px) */
- Adjusted padding
- Smaller text sizes
- Optimized touch targets
```

## Button States

### Normal
- Background: white
- Border: #d1d5db (light gray)
- Color: #374151 (dark gray)

### Hover
- Background: #f3f4f6 (lighter)
- Border: #9ca3af (darker)

### Disabled
- Opacity: 0.5
- Cursor: not-allowed

### Active (Validate)
- Background: Gradient (#667eea вҶ’ #764ba2)
- Color: white
- No border

## Interactive Elements

### Textarea
- Font: Monaco monospace
- Focus: Purple border + glow effect

- Custom scrollbar styling

### Field Cards
- Hover: Box shadow + border change

- Left accent border: Purple вҶ’ Pink on hover

- Hover: Subtle shadow
