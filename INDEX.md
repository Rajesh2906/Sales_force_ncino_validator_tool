## ✨ Feature Overview

### This app helps you:

✅ **Validate** nCino Forms Gen syntax
✅ **Visualize** form structure in human-readable format
✅ **Identify** syntax errors with line numbers
✅ **Get suggestions** for fixing errors

✅ **Understand** form structure at a glance
✅ **Load examples** to learn quickly
✅ **Work offline** (no internet needed)

## 💡 Key Concepts

### What is nCino?
nCino is a cloud banking and lending software platform.

### What is Forms Gen?
Forms Gen is nCino's form generation feature.

### What does this validator do?
It checks that your Forms Gen syntax is correct and displays the structure visually.

### How does it work?
1. You paste syntax
2. App validates it against rules
3. Shows errors with suggestions
4. Displays human-readable format
5. All happens instantly in your browser
